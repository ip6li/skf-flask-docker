# Docker based SKF

This creates a minimal working OWASP security Knowledge Framework installation
on Docker.

# Instructions

Check Dockerfile for entries

    #ARG SOURCE=blabla1337
    ARG SOURCE=ip6li

please select which meets your intentions.
